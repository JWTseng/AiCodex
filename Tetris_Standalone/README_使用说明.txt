俄罗斯方块游戏 - 独立版本使用说明
========================================

📁 文件说明：
- index.html          - 主游戏文件
- tetris.js           - 游戏逻辑
- audio.js            - 音频处理
- styles.css          - 样式文件（已优化为响应式设计）
- clear-scores.html   - 清除分数页面

🚀 启动方法：

Windows用户：
1. 双击运行 "start_windows.bat"
2. 在浏览器中访问 http://localhost:8000
3. 开始游戏！

macOS/Linux用户：
1. 双击运行 "start_mac_linux.sh" 或在终端中运行 ./start_mac_linux.sh
2. 在浏览器中访问 http://localhost:8000
3. 开始游戏！

✨ 新功能 - 响应式设计：
- 游戏界面会自动适配不同屏幕分辨率
- 支持桌面、平板、手机等设备
- 无需滚动即可完整显示游戏内容
- 自动缩放以适应屏幕大小

⚠️ 注意事项：
- 需要安装Python（Windows）或Python3（macOS/Linux）
- 确保8000端口未被占用
- 游戏需要现代浏览器支持（Chrome、Firefox、Safari、Edge）

🎮 游戏操作：
- 方向键：移动方块
- 空格键：快速下落
- P键：暂停/继续
- R键：重新开始

📞 技术支持：
如果遇到问题，请检查：
1. Python是否正确安装
2. 防火墙是否阻止了8000端口
3. 浏览器是否支持HTML5

祝您游戏愉快！
